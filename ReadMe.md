# Minesweeper API

API для игры "Сапёр" — серверная часть приложения, реализующая логику игры и REST API.

## Документация API

API доступно по адресу:  
👉 **https://minesweeper-test.studiotg.ru/swagger/**

Там вы найдете:
- Описание всех эндпоинтов
- Примеры запросов и ответов
- Интерактивную консоль для тестирования

## Сборка проекта

### Локальная сборка с Java
```bash
# Сборка JAR файла
./gradlew bootJar -x test

# Запуск приложения
java -jar build/libs/minesweeper-0.0.1-SNAPSHOT.jar
```

### Сборка с Gradle
```bash
# Очистка и сборка
./gradlew clean build -x test

# Запуск через Gradle
./gradlew bootRun
```

### Сборка с Docker
```bash
# Сборка образа
docker build -t minesweeper .

# Запуск контейнера
docker run -p 8080:8080 minesweeper:latest
```

## Доступ из интернета
После локального запуска приложение доступно только на вашем компьютере.
Чтобы сделать его доступным из интернета (например, для тестирования с мобильного устройства или для интеграции с фронтендом), используйте localhost.run.

### Инструкция для localhost.run
#### 1.Сгенерируйте SSH-ключ (если ещё нет):
```bash
ssh-keygen -t rsa
```
Просто нажимайте Enter для сохранения в стандартную папку (можно без пароля).
#### 2.Запустите туннель (приложение уже должно быть запущено локально на порту 8080):
```bash
ssh -R 80:localhost:8080 localhost.run
```
